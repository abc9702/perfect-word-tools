## Интерактивная карта для отслеживания активностей в игре Perfect World.

### Описание проекта
Интерактивная веб-карта для отслеживания активностей в игре Perfect World.

Все данные сохраняются автоматически в LocalStorage

Возможен иморт/экспорт данных (JSON).

### Основные функции
- Левый клик по точке - открыть информацию/отметить
- Масштабирование и перетаскивание карты - мышь/жесты
- Отображение - управление отображением через боковое меню
- Категории - быстрое отображение через иконки на боковой панели
- Фильтры - настройка отображения для категорий боковой панели
- Поиск - выбор и отображение активности на карте
- Редактирование - возможность редактирования описания в окне информации о точке на карте

### Мобильная версия
- Свайп от края - открыть/закрыть меню
- Pinch-жест - масштабирование карты
- Касание точки - взаимодействие

## Структура данных
- Город → Категория → Активность → Точки

### Описание JSON-файла конфигурации

#### Структура данных
```json
{
  "cities": [
    {
      "name": "string",
      "categories": [
        {
          "name": "string",
          "activities": [
            {
              "name": "string",
              "icon": "string",
              "show": boolean,
              "points": [
                {
                  "x": number,
                  "y": number,
                  "height": number | null,
                  "label": string | null,
                  "description": string | null,
                  "collected": boolean
                }
              ]
            }
          ]
        }
      ]
    }
  ]
}
```
